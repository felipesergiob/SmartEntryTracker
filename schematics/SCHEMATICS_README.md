# Esquemáticos e Fotos do Projeto

## Esquemático Digital

O arquivo `digital_circuit.png` mostra o diagrama esquemático do circuito. 

**Observações importantes:**

- O sensor mostrado no esquemático é **fictício**, usado apenas para simular o sensor de presença real
- O sensor real utilizado no projeto é da série **MH-SR602** (ou similar da série MH)
- A **protoboard foi abstraída** no esquemático digital para facilitar a visualização das conexões
- O esquemático mostra as conexões lógicas entre os componentes
